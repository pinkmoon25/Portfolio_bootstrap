## Built With

- HTML 
- Bootstrap

## Live Demo link

[portfolio](https://pinkmoon25.github.io/Portfolio_bootstrap/)


To get a local copy up and running follow these simple example steps.

### Prerequisites
have a working git on your localmachine.

## Install
To get a working local copy up and running follow these simple steps
- open terminal
- clone this project with the command:

```
$ git clone https://github.com/PinkMoon25/Portfolio_bootstrap.git
```
- Finally, run the Live Server in Visual Studio Code see the web app in your local browser.

### Usage
For anyone that wants to learn html & css & bootstrap
